# TODO — 作業タスク管理

このファイルは作業タスクを「MUST / SHOULD / WANT」で管理するための公式チェックリストです。
AIエージェントは必ずこのファイルを参照し、下記のルールに従ってください。

---

## 運用ルール（Agent向け）

- **MUST**: 必須の作業。作業完了時にすべてチェックされていなければ作業を中断し、ユーザーに確認してください。
- **SHOULD**: 取り組むべき作業。MUST と同様に、作業完了時には全てチェックを付ける必要があります。
- **WANT**: 任意の追加改善。未チェックでも構いません。
- 新しいタスクを追加する場合、行頭に `[MUST]` / `[SHOULD]` / `[WANT]` を付け、同時に `manage_todo_list` ツールで機械的に反映してください（ツールとファイルの両方を更新します）。
- 作業が完了したら、MUST と SHOULD がすべてチェック済みであることを確認してください。全てチェックされていれば、`MUST` と `SHOULD` のセクションをこのファイルから削除してください（WANT は残して構いません）。未チェックが残っている場合は作業を中断してユーザーに報告してください。
 - 作業が完了したら、MUST と SHOULD がすべてチェック済みであることを確認してください。完了したタスク行だけを削除し、セクション見出し（`MUST` / `SHOULD` / `WANT`）は必ず残してください。未チェックが残っている場合は作業を中断してユーザーに報告してください。

---

## フォーマット

- 各タスクは次の形式で記述してください:
  - `- [ ] [MUST] タイトル — 簡単な説明`
  - `- [x] [SHOULD] タイトル — 完了済みの表示`
